Carts Resource

This is a top level description of the cart resource

/carts get post

get

get all my carts

post

Create a new cart.

get /carts

get all my carts

  • Request

Query Parameters

  • Query: (string)

    query on an indexed field of the cart

    Example:

    name="A cart"

post /carts

Create a new cart.

  • Request
  • Response

Body

Type: application/json

HTTP status code 200

A cart was created.

put

delete

Delete the cart

get /carts/{id}

Retrieve a specific cart

  • Request
  • Response

URI Parameters

  • id: required (string - minLength: 1 - maxLength: 10)

    cart identifier

HTTP status code 200

Body

Type: application/json

Example:

{
  "data": {
    "id": "...",
    "name": "My House",
  },
  "success": true,
  "status": 200
}

put /carts/{id}

  • Request

URI Parameters

  • id: required (string - minLength: 1 - maxLength: 10)

    cart identifier

Body

Type: application/x-www-form-urlencoded

Form Parameters

    name of the cart

    Example:

    Naruto Uzumaki's House
  • active: (boolean)
